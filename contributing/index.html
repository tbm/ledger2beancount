Bugs and contributionsÔÉÅ

If you find any bugs in ledger2beancount or believe the conversion from ledger to beancount could be improved, please open an issue. Please include a small test case so we can reproduce the problem.

If you'd like to contribute code to ledger2beancount, please submit a pull request on GitHub or send a git format-patch patch (series) to the authors via email.

Please make sure you add a test case under tests and update the documentation (docs/, and possibly README.md and examples/illustrated.md).

You can run the test suite with make test.
